simple MEMO

Vue App May. 2023 Frontend Developer

功能簡介

  1. 使用 Vue 3.0 開發，依據功能架構拆分元件，搭配 Pinia、Vue Router，將元件共用狀態抽離，並為每個 Task 清單建立路由，可根據網址輕易切換頁面，方便編輯。
  2. 使用 Vue Use 提供的 focus、onClickOutside 處理元素焦點，提升使用者體驗。
  3. 搭配 Vue.draggable.next 套件，建立拖拉功能，能隨意將 MEMO 或是 Task 清單拖曳到想排序的位置。
  4. Tailwind CSS 快速建立 UI 樣式。
  5. 將 MEMO 列表儲存在 localStorage，網頁重新整理也不怕丟失。
